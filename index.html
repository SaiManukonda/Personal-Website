Safe Search


Flask, HTML/CSS/JS, Bootstrap

Safe Search tells its users when it is safe to visit a certain place. This app was built specifically for covid, a time when it is best to avoid big crowds. Used Flask and Google Places API for the backend. Used Bootstrap for the front end. The app fetches data from different sources and analyzes when a certain place will not be busy and displays it to the user.

Check it out!

Mirror Dashboard


Python, HTML, google-cloud, raspberry-pi

Mirror Dashboard allows its users to be constantly updated about current events despite limited connectivity as a result of network congestion. Mirror Dashboard checks for internet access periodically and updates the site’s information from a Google Cloud Compute Engine VM that scrapes the important content. The site can be accessed through the raspberry-pi WIFI network.

Check it out!
